EPICS BASE
==========
download and unpack http://www.aps.anl.gov/epics/download/base/baseR3.14.12.4.tar.gz to /usr/local/epics
install compiler: sudo apt-get install g++
install readline library: sudo apt-get install libreadline-gplv2-dev
build EPICS base in /usr/local/epics/base-VERSION/: make > make.log 2>&1


LABVIEW
========
install needed fonts
  sudo apt-get install xfonts-75dpi-transcoded xfonts-100dpi-transcoded
  restart
install LabVIEW

CA LAB
======
download and unpack http://www-csr.bessy.de/control/SoftDist/CA_Lab/caLab_VERSION.zip to /usr/locat/calab
build CA Lab:
   g++ -I/usr/local/epics/base-VERSION/include -I/usr/local/epics/base-VERSION/include/os/Linux -I/usr/local/natinst/LabVIEW-VERSION/cintools -O0 -g3 -Wall -c -fmessage-length=0 -MMD -MP -MF"caLab.d" -MT"caLab.d" -o "caLab.o" "/usr/local/calab/src/caLab.cpp"
   g++ -L/usr/local/epics/base-VERSION/lib/linux-x86 -L/usr/local/natinst/LabVIEW-VERSION/cintools -shared -o "libcalab.so"  ./caLab.o -lca -lCom
   
CONFIG SYSTEM
=============
set up library path:
	 cd /etc/ld.so.conf.d
	 sudo vi epics.conf => /usr/local/epics/base-VERSION/lib/linux-x86/
	 sudo vi calab.conf => /usr/local/natinst/LabVIEW-VERSION/cintools/
                      => /usr/local/calab/
	 sudo ldconfig
restart computer

LAST ADAPTATIONS
================
set enviroment variables EPICS_CA_ADDR_LIST and EPICS_CA_AUTO_ADDR_LIST
start Soft IOC shell: /usr/local/epics/base-VERSION/bin/linux-x86/softIoc -D /usr/local/epics/base-VERSION/dbd/softIoc.dbd -d /usr/local/calab/demo/db/demo.db
start labVIEW
open '/usr/local/calab/VIs/Private/CaLabGet_Main.vi' and edit all 'Library name or path' of 'Call Library Function Nodes' (double click on it) => change entry to '/usr/local/calab/libcalab.so'.
repeat this step '/usr/local/calab/VIs/Private/CaLabPut_main.vi' and '/usr/local/calab/VIs/CaLabEvent.vi'.

Start Demo Soft IOC
====================
/usr/local/epics/base-VERSION/bin/linux-x86/softIoc -D /usr/local/epics/base_VERSION/dbd/softIoc.dbd -d /usr/local/calab/demo/db/demo.db

enjoy  CA Lab
